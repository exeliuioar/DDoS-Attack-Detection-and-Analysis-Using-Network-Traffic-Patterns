# DDoS Attack Detection and Analysis Using Network Traffic Patterns

Simple Python tool to detect DDoS attacks from network traffic.

## Features
- Live capture or PCAP file analysis
- Statistical thresholds
- Source IP / port entropy detection
- Basic dashboard

## Installation
```bash
git clone https://github.com/exeliuioar/DDoS-Attack-Detection-and-Analysis-Using-Network-Traffic-Patterns.git
cd DDoS-Attack-Detection-and-Analysis-Using-Network-Traffic-Patterns
pip install -r requirements.txt
